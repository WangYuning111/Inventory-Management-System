📥 Import Inventory Data

Import items from JSON or CSV format. This will replace all existing data.

📋 Format Guide:

JSON should be an array of items or an object with items array.

{{ '{' }}"id": "ITEM001", "name": "Wireless Mouse", "category": "Electronics", "quantity": 10, "price": 29.99, "supplierName": "TechCorp"{{ '}' }}

CSV must include headers: ID, Name, Category, Quantity, Price, Supplier

ID,Name,Category,Quantity,Price,Supplier
